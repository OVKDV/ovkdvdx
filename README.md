# DXF to SVG/3D Converter

## Структура проекта:
- index.php - главная страница
- config.php - конфигурация
- css/style.css - стили
- js/main.js - основная логика
- js/converter.js - конвертер DXF
- js/3d-viewer.js - 3D визуализация
- js/upload-handler.js - обработка загрузки
- components/ - PHP компоненты
- api/ - API endpoints
